# MobiCova USSD Service — Menu Walkthrough

**Prepared for:** Africa's Talking (USSD Service Application)
**Service name:** MobiCova Digital Health
**USSD shortcode:** `*XYZ#` *(placeholder — replace with the dedicated shortcode assigned by AT)*
**Document version:** 1.0 — 11 June 2026

> **Change control:** This menu structure and flow are fixed as documented. MobiCova will not alter the menu, options, ordering, or text without prior **written notice to Africa's Talking.**

---

## 1. Technical & session conventions

| Item | Specification |
|---|---|
| Session model | `CON` = menu continues (awaits input); `END` = session terminates |
| Screen limit | Each screen kept within **182 characters** |
| Input | Single digit/number per screen unless capturing an ID, PIN, or free text |
| Navigation | `0` = Back one level · `00` = Main menu · `98` = Next page (where lists are long) |
| Session timeout | Per AT default (member re-dials `*XYZ#` to resume) |
| Authentication | Member auto-recognised by calling number (MSISDN). A **4-digit PIN** is required before any sensitive action (membership/cover details, telemedicine, PIN change). Public actions (symptom check, find care) need no PIN. |
| Language | English (v1.0) |

## 2. Menu tree (overview)

```
*XYZ#  (dial)
│
├─ [Recognised member] → MAIN MENU
│   1. My Membership & Cover        → PIN required
│   2. Request Telemedicine          → PIN required
│   3. AI Symptom Check              → no PIN
│   4. Find Care Near Me             → no PIN
│   5. Help & Settings               → mixed
│
└─ [Unrecognised number] → WELCOME MENU
    1. Register (link Membership ID to this phone)
    2. Find Care Near Me
    3. About MobiCova
```

## 3. Screen-by-screen walkthrough

### 3.0 Entry — dial `*XYZ#`

**A. If the calling number IS linked to a member:**
```
CON Welcome to MobiCova, [FirstName].
1. My Membership & Cover
2. Request Telemedicine
3. AI Symptom Check
4. Find Care Near Me
5. Help & Settings
```

**B. If the calling number is NOT linked to a member:**
```
CON Welcome to MobiCova.
This number isn't linked to a membership.
1. Register
2. Find Care Near Me
3. About MobiCova
```

---

### 3.1 Register (from Welcome → 1)

```
CON Enter your MobiCova Membership ID:
```
↳ member types ID (e.g. `MC100245`)
```
CON Create a 4-digit PIN:
```
↳ member types PIN (e.g. `1234`)
```
CON Confirm your 4-digit PIN:
```
↳ member re-types PIN
```
END Success! Membership MC100245 is now linked to this phone.
Dial *XYZ# anytime to access your cover.
```
*Failure cases:* unknown/duplicate ID → `END Membership ID not found or already linked. Contact support on 0700-MOBICOVA.` · PIN mismatch → `END PINs did not match. Please dial *XYZ# and try again.`

### 3.4 AI Symptom Check (Main → 3) — *no PIN*

```
CON AI Symptom Check
This gives guidance only, not a diagnosis.
1. Fever
2. Cough / cold
3. Stomach / diarrhoea
4. Headache
5. Other
```
↳ select symptom (e.g. `1`) →
```
CON How long have you had it?
1. Less than 1 day
2. 1-3 days
3. More than 3 days
```
↳ select →
```
CON Any severe signs? (chest pain, trouble breathing, fainting, blood)
1. Yes
2. No
```

**If `1` (severe signs):**
```
END This may be urgent. Please seek care now.
Dial *XYZ# > Request Telemedicine, or go to your nearest clinic. Emergency: call 112.
```

**If `2` (no severe signs) — example output:**
```
END Guidance: Rest, fluids, monitor temperature. If it worsens or lasts >3 days, request a teleconsult via *XYZ# > 2. Generated by MobiCova AI Health Assistant.
```
> *Note for AT:* triage paths are menu-driven; the response text is produced server-side by MobiCova's AI Health Assistant within the USSD response window. It provides advice and escalation only — never a diagnosis.

## 4. Error & edge-case handling (applies to all screens)

| Scenario | System response |
|---|---|
| Invalid menu choice | `CON Invalid choice. Please try again.` *(menu re-displayed)* |
| Wrong PIN (attempts 1–2) | `CON Incorrect PIN. Attempts left: [n]. Re-enter PIN:` |
| Wrong PIN (3rd failure) | `END Too many wrong attempts. PIN locked. Call 0700-MOBICOVA to reset.` |
| Member inactive / lapsed cover | `END Your cover is currently inactive. Contact your HR or 0700-MOBICOVA.` |
| Backend unavailable | `END Service is temporarily unavailable. Please try again shortly.` |
| Session timeout | Session ends; member re-dials `*XYZ#` |
